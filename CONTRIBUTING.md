# Contributing to qiita-content

- https://qiita.com/Qiita/items/32c79014509987541130
- https://github.com/increments/qiita-cli

## How To Init

```Shell
npm install @qiita/qiita-cli --save-dev
```

```Shell
npx qiita version
```

```Shell
npx qiita init
```

### トークンを作成してログイン

```Shell
npx qiita login
```

### 記事のプレビュー

```Shell
npx qiita preview
```
